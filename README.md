# Riyadh Luxury Curtains Landing Page

Premium Next.js single-page site designed to convert Google My Business ad traffic into calls for a bespoke curtain business in Riyadh, Saudi Arabia. Copywriting and visuals target high-end residential and commercial clients, highlighting rapid consultation, luxury fabric collections, and concierge installation.

## Tech Stack

- Next.js 14 (App Router, TypeScript)
- React 18
- `next/font` with Google Fonts
- Handcrafted responsive CSS
- React Icons

## Getting Started

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to view the site.

## Production Build

```bash
npm run build
npm start
```

## Linting

```bash
npm run lint
```

## Deployment

Optimized for Vercel. After building locally, deploy with:

```bash
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-06b6c302
```
